## 五、影像资料（Gallery）

**Section ID：** `#gallery`

**Section 标题：** 影像资料

### 主展示区：轮播画廊

**交互形式：** 3D 卡片轮播，自动播放
- 当前展示照片：正中央，最大，完整显示
- 左侧：上一张，缩小 + 后移（scale + translateZ），半透明
- 右侧：下一张，缩小 + 后移，半透明
- 循环展示，无限轮播
- 建议使用 [Swiper.js](https://swiperjs.com/) 的 `coverflow` 效果实现

**每张照片结构：**
- 照片本体（统一16:9比例）
- 下半部分叠加从透明到黑色的渐变遮罩（`linear-gradient(to top, rgba(0,0,0,0.7), transparent)`）
- 遮罩上方文字：
  - 活动名称（白色，中号加粗）
  - 活动时间（白色，小号，透明度 80%）

## 九、设计规范建议

> 供 AI 建站工具参考，非强制，可根据整体风格调整。

### 字体

| 用途 | 建议 |
|------|------|
| 中文 | OPPO Sans |
| 英文 / 数字 | Inter / Space Grotesk |

### 动效原则

- 页面滚动进入动画：元素从下方淡入（`opacity 0→1` + `translateY 20px→0`）
- 卡片 Hover：轻微上浮（`translateY -6px`）+ 阴影扩大，过渡 `0.3s ease`
- 轮播切换：缓动曲线（`ease-in-out`），时长 `0.5–0.8s`
- 整体风格：克制、高级，避免过于花哨

### 技术栈建议（供参考）

- 框架：Next.js / Astro（静态优先）
- 样式：Tailwind CSS
- 轮播：Swiper.js（`coverflow` 模式）
- 动效：Framer Motion 或 AOS.js（滚动动画）
- 图片：统一压缩为 WebP，懒加载
